Delete Prefs in Final Cut Pro X

HOW TO DELETE FCPX PREFERENCES

• May 23rd, 2013 • Connor Crosby

TUTORIAL DETAILS

• Difficulty: Easy • Minimum Requirements: Final Cut Pro 10.0.0 Does Final Cut Pro X keep crashing or are you getting weird bugs you haven’t run into before? Deleting the preferences may help fix it. USING FINDER To delete the FCPX preferences using Finder, go to User > Library > Preferences > com.apple.FinalCut.plist and delete that file. Also go to User > Library > Preferences > com.apple.FinalCut.LSShardFileList.plist and delete that too.

USING PREFERENCE MANAGER

If you aren’t in the mood to fish through Finder, you can also use a free program called Preference Manager by Digital Rebellion. Preference Manager allows you to manage preference files for Final Cut Studio, Final Cut Express, Final Cut Pro X, Avid Media Composer, Avid Symphony and Adobe Premiere Pro for Mac.

Open Preference Manager and select Final Cut Pro X.

Now press the “Trash” button. This will delete those files above for you. Preference Manager also enables you to back up your preferences in case you ever need to delete them. Using either Finder or Preference Manager will hopefully take care of the issue and you can get back to editing.
